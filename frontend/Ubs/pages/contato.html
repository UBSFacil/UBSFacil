Perguntas frequentes

O UBS Fácil é gratuito?
Sim, o UBS Fácil é 100% gratuito para todos os cidadãos brasileiros. A plataforma é mantida pelo Ministério da Saúde e não possui nenhum tipo de cobrança.
Como saber se um medicamento está disponível?
Use a busca na página de Medicamentos. Informe o nome do medicamento para ver as unidades que têm o produto em estoque.
Preciso de conta para usar o UBS Fácil?
Para funções básicas como buscar medicamentos e encontrar unidades no mapa, não é necessário login. Para funcionalidades personalizadas, você precisará se autenticar via Gov.br.
Os dados de saúde são protegidos?
Sim. Todos os dados pessoais são tratados em conformidade com a LGPD (Lei nº 13.709/2018). A autenticação é feita exclusivamente via Gov.br.
Como reportar uma informação incorreta?
Use o formulário acima selecionando "Unidade de saúde desatualizada". Nossa equipe irá verificar e corrigir junto às secretarias de saúde responsáveis.
